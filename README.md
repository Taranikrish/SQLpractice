# SQL Mini Project – Student Management System

This project demonstrates my hands-on practice of SQL concepts including table creation, data insertion, and query execution.

## Concepts Used
- CREATE TABLE
- INSERT INTO
- SELECT queries
- JOIN operations
- GROUP BY and aggregation
- Subqueries

## Description
A simple database system was created to manage students and their respective departments. Various queries were written to extract meaningful insights such as top performers, department-wise averages, and filtered results.

## Outcome
This project helped me understand how SQL is used to manage structured data and perform analysis using queries.
